Front-end нотатки

Питання на співбесіді

  1. Що таке HTML

    Hyper Text Markup Language - мова розмітки гіпертексту, для перегляду вебсторінок у браузері

  2. Що таке парні та непарні теги

    Парні теги потрібно закривати, інакше браузер може некорректно відобразити їх вміст: p, div, a, h1
    Непраня теги не мають тегу, що закриваеться і не мають вмісту: img, br, hr

  3. Що таке блочні та лінійні елементи

    Блочні займають всю ширину сторінки: div (король блочних елементів), p, h1..h6, table
    Лінійні займають на сторінці лише всою ширину: span (король лінійних елементів), em, strong, img, button

  4. Можуть запитати про формати фото

  5. Можуть попросити щось написати в codepen

  6. Які семантичні теги html5 знаєш

    headern footer, aside, article, nav

Гарячі клавіші

  • ! tab або enter - дефолтна розмітка
  • .text створює div з класом text
  • footer.text створює footer з класом text

Правила верстки

  1. даємо назви (class) всім елементам, однаковим елементам можна давати ідентичні назви
  2. завертаємо весь сайт в div з класом wrapper
  3. кожну секцію, крім main (section, article, footer, header) завертаємо в div з класом container
  4. якщо в проекті було десь використано max width, далі використовуємо лише max width; max width - це desktop first верстка; min width - це mobile first верстка
